O que é Kotlin?
- Criada pela JetBrains em 2011, mantida pela Kotlin foundation (parceria com o Google);
- Linguagem multiplataforma, orientada a objetos e funcional;
- Gratuito e código aberto;
- Nome em homenagem a ilha russa, Kotlin

Estrutura da Linguagem
- Não utiliza ponto e virgula
- Não possui operador ternario tradicional
- Type safe e null safe (motivo pela grande adesão) evita erros de execução
- Inferencia de tipos

Aplicabilidade
- Android nativo
- Mobile multiplataforma (KMM)
- Frontend web com kotlin/js
- Frameworks ex: kvison e fritz2
- Backend com target node.js do kotlin/js

Vantagens kotlin
- Segura, estruturada, menos verbosa, mais otimizada, grande relevancia no mercado
- Interoperabilidade com o java; utilizar com a linguagem java e kotlin em uma aplicação, sem necessidade de imigrar a aplicação inteira para substituir para kotlin
- Smart casts e null safety
- Android é kotlin first

Desvantagens
- desempenho (um pouco mais lento)


Kotlin + Android
- 2016: primeira versão estavel do Kotlin Liberada
- 2017: Google anuncia suporte a Kotlin para desenvolvimento em android
- 2018: segunda linguagem preferida pelos desenvolvedores segundo stack overflow
- 2019: Google anuncia que Kotlin é a nova linguagem oficial para desenvolvimento android
(documentação, apps e tudo do google seria feito em Kotlin)
