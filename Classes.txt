=> Hierarquia de classes é uma árvore de classes em que as classes filhas herdam a funcionalidade das classes pai. Propriedades e funções são herdadas por subclasses;
criar uma classe abstract em que algumas funcionalidades podem ser implementados pelas subclasses. 
*uma classe abstract não pode ser instanciada;
=> a palavra-chave override para modificar propriedades e funções em subclasses;
=> a palavra-chave super para referenciar funções e propriedades na classe pai;
=> uma classe open => permite que uma subclasse possa herdar dela
=> uma propriedade private, só pode ser usada dentro da classe;
=> a construção with permite fazer várias chamadas na mesma instância do objeto
